## Supported Dexes

- Pump AMM
- Raydium V4
- Raydium CPMM
- Raydium CLMM
- Meteora DLMM
- Meteora Dynamic AMM
- Meteora DAMM V2


## Getting Started

### Prerequisites

- Rust and Cargo installed
- Solana wallet with SOL for transactions
- Access to Solana RPC endpoints

### Installation

1. Clone and run:
   ```bash
   cargo run --release --bin solana-onchain-arbitrage-bot -- --config config.toml
   ```
